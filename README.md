This specific version of TWRP tree is built for **A266BXXS4AYGA.**

**To setup your build enviorment**, follow this [**guide.**](https://github.com/minimal-manifest-twrp/platform_manifest_twrp_aosp/blob/twrp-12.1/README.md)

> To build TWRP using this device trees, run these commands:

```bash
export ALLOW_MISSING_DEPENDENCIES=true
. build/envsetup.sh
lunch twrp_a54x-eng
mka recoveryimage
```
